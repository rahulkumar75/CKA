Client and server have -> Pub Cert & Private key

Certificate -> *.cert and *.pem
Private key -> *.key

Issue Cert -> For new user to grant perm of cluster (Exm)

Exec inside pods:
1. Gen Private key -> openssl genrsa  -out rahul.key 2048
---
# ls | grep .key
rahul.key 
cat rahul.key 


2. CSR content generated from OpenSSL. -> openssl req -new -key rahul.key -out rahul.csr -subj "/CN=rahul"
---
# ls | grep .csr
rahul.csr
cat rahul.csr -> Plain file


3. Encryption CSR file - Encoded:
cat rahul.csr | base64 | tr -d "\n" -> copy and then Paste output inside request in (rahul-csr.yaml) file.


4. Apply CSR yaml file


5. Approve -> k certificate approve rahul-csr


6. kubectl describe csr <name-of-csr>


7.Share Certificate(encoded) with user (file created) -> k get csr adam-csr -o yaml > adam-issuecert.yaml


8. Get Certificate -> kubectl get csr rahul-csr -o jsonpath='{.status.certificate}' | base64 -d

9. To decode the adam.key 
certificate(copy request value from issued-cert.ymal) -> 
Decode By using Private key -> echo “paste-key-here” | base64  —d
